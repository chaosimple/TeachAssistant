# 课堂互动系统

一个纯前端实现的课堂互动与成绩管理系统，支持随机点名、表现评分、成绩统计等功能，数据通过 Excel 文件实现便携式存储。

## 功能特点

### 🎯 课堂互动
- **权重随机点名**：互动少的学生有更高概率被选中，自动趋向公平
- **滚动动画**：点名时带有老虎机式滚动效果，可关闭
- **烟花庆祝**：选中学生后触发烟花彩带动画
- **搜索点名**：支持按学号、姓名、拼音首字母搜索学生
- **星星评分**：支持半星评分（0.5-5星，对应10-100分）
- **快捷操作**：一键设置缺勤(0分)、请假(60分)

### 👥 学生管理
- Excel 批量导入学生名单
- 手动添加/编辑/删除学生
- 显示学生互动次数统计

### 📊 成绩管理
- 按周记录课堂表现成绩
- 动态周数表格（根据学期设置自动计算）
- 点击单元格可直接编辑成绩
- 导出成绩为 Excel 文件

### 📝 历史记录
- 查看所有互动记录
- 按日期筛选
- 支持修改评分、删除记录
- 导出互动记录为 Excel

### 🏆 数据统计
- **本周统计**：平均分排行榜 TOP5、互动频次图、统计摘要
- **本学期统计**：学期排行榜 TOP5、学期频次图、总体统计
- 实时刷新，切换页面自动更新

### ⚙️ 系统设置
- 自定义课程名称
- 设置学期起止日期（自动计算周数）
- 开关动态点名效果

### 💾 数据持久化
- 本地 localStorage 存储
- 完整数据备份/恢复（Excel 多工作表）
- 未保存提醒，防止数据丢失
- 首次使用引导弹窗

## 快速开始

1. 用浏览器打开 `index.html` 文件
2. 首次使用会显示欢迎弹窗，可选择导入数据或手动设置
3. 在「设置」页面配置课程名称和学期时间
4. 导入学生名单或手动添加学生
5. 开始课堂互动

## Excel 文件格式

### 学生名单导入格式

| 学号 | 姓名 |
|------|------|
| 2024001 | 张三 |
| 2024002 | 李四 |

- 第一行为表头（可省略）
- 支持 `.xlsx` 和 `.xls` 格式

### 完整数据备份格式

导出的备份文件包含 4 个工作表：

| 工作表 | 内容 |
|--------|------|
| 系统设置 | 课程名称、学期时间、备份时间 |
| 学生名单 | 学号、姓名、互动次数 |
| 互动记录 | 日期、时间、学号、姓名、评分、周次 |
| 成绩数据 | 学号、姓名、各周成绩、平均分、评分次数 |

## 随机点名算法

采用**权重随机选择算法**，核心公式：

```
权重 = 最大互动次数 - 当前学生互动次数 + 1
```

- 互动次数少的学生权重更高，被选中概率更大
- 随着学期进行自动平衡，趋向公平
- 即使多次被点名，权重最低为 1，不会完全失去机会

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **Excel处理**：SheetJS (xlsx) - 本地化部署
- **数据存储**：localStorage

## 使用建议

1. **定期备份**：点击「备份数据」按钮导出 Excel 文件
2. **学期初设置**：先在设置页面配置学期时间，确保周数计算正确
3. **数据迁移**：通过导入/导出 Excel 文件实现跨设备使用
